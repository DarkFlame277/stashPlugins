# Stash Plugins

This is a repo for my [Stash](https://github.com/stashapp/stash) plugins (just one right now).

To use them, add "[https://darkflame277.github.io/stashPlugins/main/index.yml](https://darkflame277.github.io/stashPlugins/main/index.yml)" as a source in your Available Plugins.

## [Gallery-DL Metadata Importer](https://github.com/DarkFlame277/stashPlugins/tree/main/plugins/gallerydl-metadata-importer)
This plugin imports metadata from [Gallery-DL](https://github.com/mikf/gallery-dl) JSON files into Stash images and scenes.
